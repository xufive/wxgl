# 光照情景模式

WxGL提供多种光照情景模式。配合环境光（ambient）、杂散光（stray）、粗糙度（roughness）、金属度（金属度）、透光度（pellucidness）、光洁度（shininess）等参数，可模拟不同的质感。

* [环境光照情景模式](https://xufive.github.io/wxgl/api/light.html#wxglbaselight)
* [太阳光照情景模式](https://xufive.github.io/wxgl/api/light.html#wxglsunlight)
* [定位光照情景模式](https://xufive.github.io/wxgl/api/light.html#wxgllamplight)
* [户外光照情景模式](https://xufive.github.io/wxgl/api/light.html#wxglskylight)
* [球谐光照情景模式](https://xufive.github.io/wxgl/api/light.html#wxglspherelight)
