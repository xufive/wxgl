# 3D绘图工具包（wxgl.glplot）

wxgl.glplot包含了功能完备的绘图函数，这些函数的命名和使用方式都非常类似matplotlib.pyplot子模块。正如matplotlib.pyplot被简写为plt那样，glt被视为wxgl.glplot的别名。

```python
import wxgl.glplot as glt
```

* glt.figure()
* glt.subplot()
* glt.show()
* glt.savefig()
* glt.line()
* glt.point()
* glt.surface()
* glt.quad()
* glt.mesh()
* glt.title()
* glt.colorbar()
