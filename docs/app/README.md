# 与wxPython集成

场景类[wxgl.Scene](https://xufive.github.io/wxgl/api/scene.html#wxglscene)是wx.glcanvas.GLCanvas的派生类。作为容器，场景通过调用[add_region](https://xufive.github.io/wxgl/api/scene.html#wxglsceneadd_region)方法添加视区[wxgl.Region](https://xufive.github.io/wxgl/api/region.html#wxglregion)，所有的模型都绘制在视区内。
